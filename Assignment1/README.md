# Reinforcement Learning - Assignment 1

This repository contains the implementation and analysis for Assignment 1, covering Markov Decision Processes (Grid World) and Reinforcement Learning algorithms (SARSA and Q-Learning on Acrobot).

## Directory Structure

### Task 1: Grid World & Value Iteration
Located in `Task1/`

*   **`task1.py`**: A standalone Python script that implements the Grid World MDP and the Value Iteration algorithm. Running this script executes the value iteration process and displays snapshots of the value function at different iterations.
*   **`task1.ipynb`**: A Jupyter Notebook that extends the base implementation. It runs Value Iteration under various experimental settings (e.g., $\gamma=0.3$, High Penalty, High Drift) and generates the specific visualizations used in the report.
*   **`writeup.ipynb`**: A notebook containing the written answers, observations, and detailed analysis for the questions posed in Task 1.
*   **`output/`**: Directory containing the generated plots for the different MDP configurations.

### Task 2: Acrobot & RL Algorithms
Located in `Task2/`

*   **`keyboard_acrobot.py`**: A utility script that allows you to manually control the Acrobot environment using the keyboard. This is useful for building intuition about the environment's dynamics.
*   **`hyperparam_search.py`**: A script designed to perform a hyperparameter sweep (grid search) for both Q-Learning and SARSA agents. It saves the performance metrics to `outputs/`.
*   **`random_seed_sim.py`**: A script that takes the best hyperparameter configurations and runs the agents across multiple random seeds to evaluate stability and variance. Results are saved to `outputs/`.
*   **`task2.ipynb`**: The main analysis notebook for Task 2. It loads the results generated by the python scripts, visualizes learning curves, and produces the figures included in the report.
*   **`outputs/`**: Directory storing `.pkl` files containing the serialized results from the hyperparameter search and random seed simulations.

### Report
Located in `report/`

*   **`report.pdf`**: The final compiled report document.
*   **`report.typ`**: The Typst source file used to generate the PDF.
*   **`*.png`**: Various plots and visualizations referenced in the report.

## How to Run

### Prerequisites
Ensure you have the required Python dependencies installed. You can install them using:
```bash
pip install -r requirements.txt
```

### Running Task 1
1.  **Base Implementation**:
    To run the standard Value Iteration and see the value function evolution:
    ```bash
    python Task1/task1.py
    ```
2.  **Experiments & Plots**:
    Open `Task1/task1.ipynb` in Jupyter Notebook or JupyterLab. Run all cells to generate the visualizations for the different MDP scenarios (e.g., different discount factors or wind probabilities).
3.  **Analysis**:
    Open `Task1/writeup.ipynb` to read the answers to the assignment questions.

### Running Task 2
1.  **Manual Control (Optional)**:
    To interact with the Acrobot environment:
    ```bash
    python Task2/keyboard_acrobot.py
    ```
2.  **Hyperparameter Search**:
    Run this script to perform the parameter sweep. **Note:** This may take some time to complete.
    ```bash
    python Task2/hyperparam_search.py
    ```
3.  **Random Seed Simulation**:
    After the search is complete, run this script to evaluate the best agents over multiple seeds:
    ```bash
    python Task2/random_seed_sim.py
    ```
4.  **Analysis & Plotting**:
    Open `Task2/task2.ipynb` and run the cells. This notebook will load the pickle files from `Task2/outputs/` and generate the final plots used in the report.
